Simscape Multibody

The main aim of this project was to create 2-DOF spherical robot using Matlab Simscape Multibody.A simulation was created and a attempt was made to implement machine learning from sample dataset.

Obstacle avoidance and object detection using Turtlebot in ROS Gazebo

The main aim of this project was to detect and move to objects, pick them up, and drop them in the corresponding bins while avoiding obstacles.

Keypoints: ROS, Gazebo, Object Detection, SURF algorith, ROS Move_Base

Smart Grocery Machine

The aim of this project is to obtain grocery from the machine as per defined by customer. Machine releases the required amount of grocerry and an invoice is generated on completion of process.

Keypoints: Arduino, algorithm, sensors and tinkering.

Path Planning using A* and VFH.

The goal of this mini-project was to autonomously plan and execute a path for 2D robot in a stage simulator from user defined start and goal location in a given map.

A* algorithm was used as global planning while Vector Field Histogram (VFH) was used as local planning.

5-DOF Fix Arm Robot

This is my first manufactured and working Robot. It is 5 degree of freedom robot. It consits of several servo motors and Arduino Mega was used as microcontroller. All parts were manufactured on Laser cutting machine.

Hexapod Robot

The main of this mini-project was to create a model and simulate a Hexapod robot using Solidworks. This is the very first project where my passion for robotics emerged.

About Me

I am a graduate student pursuing Robotics engineering at University at Buffalo. My interests are in collaborative robotics, perception and motion planning. Robotics are no longer a mere subject of science fiction novels, movies and TV shows.They already coexist with people in various settings, albeit not in the form sci-fi stories used to depict them. I believe that they have the potential to change the world. Collaborative robots reflects the focus of the industry towards creating robots that are simpler, easier to program, and able to integrate into current processes.

I have worked on Arduino, Raspberry Pi, encoders, motors and 3D printing and laser cutting machine.

I always love to gain the knowledge from many verticles which is extremely crucial for robotics. I have sound experience of Robotic system development starting from conception to fabrication to electionics to application related programming.

Along with robotics, I always listen to music a lot. My favourite band is The Local Train. Music works like a coffee to me. I always love to go for mountain hikes. I'm die hard fan of FC Barcelona. Forca Barca!!
